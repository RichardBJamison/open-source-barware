# Open Source Barware — Download Package v1.0

Spreadsheet templates and compliance files for hospitality teams.  
**For the full Chrome inventory program** (Mac/Windows installers), go to:

**https://opensourcebarware.com/download**

---

## What's in this spreadsheet package

- `Bar-Inventory-Master.xlsx` — master inventory workbook template
- `Quick-Count-Sheet.xlsx` — count worksheet template
- `Variance-Calculator.xlsx` — variance and usage worksheet
- `Product-Database.xlsx` — starter product reference workbook
- `master-prompt.md` — optional AI assistant instructions (not required for the program)
- `LICENSE.txt` — GNU GPLv3 license text
- `NOTICE.md` — copyright and warranty notice
- `SOURCE-OFFER.md` — corresponding source access notice

## Chrome program installers (v1.0)

| Platform | File |
|---|---|
| macOS | `open-source-barware-program-mac.zip` |
| Windows | `open-source-barware-program-win.zip` |

Unzip → run `Install.command` (Mac) or `Install.bat` (Windows).  
Read `README-INSTALL.txt` inside the zip for full steps and how to clean up your Downloads folder after install.
