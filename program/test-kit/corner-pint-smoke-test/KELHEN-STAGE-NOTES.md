# Kelhen — Corner Pint smoke test (run this first)

**Use this kit before Harbor & Hearth.** One walk, one count, two fixable gaps.

## Fresh start

```bash
rm -rf ~/osb-program
cd ~/Downloads
unzip -o open-source-barware-program-mac.zip
cd open-source-barware-program
./Install.command
open http://localhost:5052/setup
```

Chrome: **Cmd+Shift+R** hard refresh.

## Files

```
KIT=~/Downloads/open-source-barware-program/test-kit/corner-pint-smoke-test
```

| Step | What | File |
|------|------|------|
| Walk | Bar name **Corner Pint** | `$KIT/walk.txt` |
| Count | Paste after map approved | `$KIT/count.txt` |

## Count step — what you'll see

- **2 gaps** (not 72): Hendricks missing, Campari surprise
- Use **Fix** column: **Counted** / **Not on shelf** / **Add to map** / **Misheard**
- Finish unlocks when gaps ≤ 2 (or you fixed them)

## Do NOT

- Paste Harbor & Hearth walk files into Corner Pint
- Merge multiple bar walks into one paste
