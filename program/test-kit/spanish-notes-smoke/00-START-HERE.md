# Spanish notes smoke kit (MX)

Walk and count files use Mexican Spanish structure words.
v1.5 UI remains English; parser accepts EN + ES structure synonyms.

1. Create a fresh bar in setup.
2. Upload `walk.txt` (or paste) → Parse.
3. Confirm stations resolve: Well / Back bar / Beer cooler, two bars.
4. Approve map, paste `count.txt`, Process.
5. Levels: one, half, full, out, 0.8 style from Spanish words.
