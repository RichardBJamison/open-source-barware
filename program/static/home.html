Optional accuracy upgrade

Weighing mode (for digital scale)

Completely optional. Most venues start with simple level counts. Enable this to enter current bottle weights instead of (or alongside) levels for much higher accuracy. Perfect transition path when you add a digital scale later.

(uses bottle full/empty weights)

Coming as add-on: official Bluetooth liquor scale for one-tap super-accurate weighing. Enable mode now to get familiar — your data stays the same if you switch back.

Weight fields appear in count views only when enabled. Levels always work as fallback.

Optional add-on

Optional AI connection

The program does not need AI to run. Walks, counts, reconciliation, and spreadsheets are built in. Connect only if you want phone invoice photos read automatically — or for upcoming wine-cooler video walks. Otherwise type invoice numbers yourself in All inputs or your workbook.
