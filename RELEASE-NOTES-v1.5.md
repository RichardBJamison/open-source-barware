**Release date:** July 10, 2026

## Summary

The July 4 free launch is now the full **v1.5** public drop. Same local-first Chrome program on Mac and Windows — no subscription, no cloud account required. Installers on the Download page are v1.5.

## What's new in v1.5

### Language / inventory notes
- **Spanish + English inventory notes** — walks and counts understand Spanish station and level words (e.g. *barra*, *pozo*, *fila*, *medio*, *tres cuartos*, *lleno*, *vacío*) alongside English
- Mixed EN/ES notes work today; **full Spanish program UI** is next (Coming soon panel)

### Team & communications
- **Employee communications board** — managers upload or paste walks and inventory notes for the house (local-only)
- **Team PIN logins** — optional; open until you create the first admin PIN

### Counting & floor work
- **Mobile count view** — large taps, presets, optional weigh mode
- **Camera barcode scan**
- **Bottle-weight table** for optional digital scale weighing
- Visual par alerts (green / yellow / red)

### Ops toolkit
- **Smart orders → purchase orders** (copy, email, CSV, receive workflow)
- **POS import** with review step (Toast, Square, CSV, etc.)
- **Recipe builder + menu costing**
- **Shift / cycle reports** — plain-English weekly story + export
- **Multi-venue + stock transfers**

### Roadmap surface
- **Coming soon** panel in Settings — full Spanish UI, Intelligent Hospitality Systems restaurant package, 2.0

## Install

1. Download Mac or Windows zip from https://opensourcebarware.com/download  
2. Unzip → run `Install.command` (Mac: right-click → Open once) or `Install.bat` (Windows)  
3. Bookmark `http://localhost:5052/` — that bookmark is the software  
4. Verify: `http://localhost:5052/ping` returns `"status": "ok"`

See `program/README-INSTALL.txt` and `program/VERSION.txt` inside the zip for full install notes.
